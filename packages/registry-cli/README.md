## Overview

Command-line interface for managing wallet entries in the PartyLayer registry. Used for validating, adding, and updating wallet metadata.

---

## Installation

```bash
npm install -g @partylayer/registry-cli
```

---

## Usage

```bash
partylayer-registry --help
```
